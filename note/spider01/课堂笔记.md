1. 爬虫的概念
   模拟浏览器发送请求获取响应

2. 数据的来源
   a. 公司自行产生
   b. 购买
   c. 免费获取
   d. 人工获取
   e. 爬虫抓取

3. 爬虫的分类
   a. 通用
   b. 聚焦

4. 爬虫的流程
   向起始url发送请求，并获取响应
   对响应进行提取
   如果提取url，则继续发送请求获取响应
   如果提取数据，则将数据进行保存

5. http和https的概念
   http 超文本 传输 协议 80
   https http + ssl 443

6. 浏览器中最终结果是多次请求对应的多次响应内容共同渲染的结果

7. HTTP重要请求头
   `cookie`,`referer`,`user-agent`

8. 重要响应头
   `set-cookie`

9. 常见的状态码
   200
   302.307
   404
   500.503

10. str和bytes类型的互相转换

   ```
   string encode() --> bytes
   bytestr decode() --> str
   ```
